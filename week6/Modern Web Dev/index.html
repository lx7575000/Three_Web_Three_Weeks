会用到的工具

HTML5 是一个充满激情的帝国！下面这些工具基本都是帝国内的明星。我们会用 github 来版本控制，chrome 开发者工具来调试代码，也会使用命令行中的高效工具 Bower 和 Gulp 来辅助开发。会共同深入研究 web component 的思想，所以下面列出了 ploymer 和 reactjs 。 Svg 的外号叫“响应式图形”，响应式开发怎么能不用。Sass 预处理是必备利器，也需要有 Browser-sync 来进行页面自动刷新。Material Design 是 UI 规范，能让没有专门学过设计的我们做出专业的界面。Yeoman 是前端老手经验总结，各种技巧的宝库。对了，课程推荐的开发机器是苹果，总感觉用 Windows 开发是件很不严肃的事情。
